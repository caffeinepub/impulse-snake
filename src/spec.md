# Specification

## Summary
**Goal:** Increase the Snake game playfield canvas size to make the gameplay area significantly larger and more prominent on screen.

**Planned changes:**
- Expand the playfield canvas dimensions to occupy more screen space
- Ensure the larger canvas remains responsive across desktop, tablet, and mobile devices
- Maintain square aspect ratio and consistent styling

**User-visible outcome:** Players will see a noticeably larger gameplay area that is more comfortable to play on, with the Snake game canvas prominently displayed instead of appearing small and centered.
